# Dotfiles

My dotfiles

## Setup

Install GNU `stow`

```sh
sudo apt install stow
```

Clone project to the home directory

```sh
git clone https://github.com/hieuvuvan/dotfiles
```

Using stow to create symbolinks to home
```sh
# Go to dotfiles
cd dotfiles
# Link git package to home
stow git 

# Other package link
```
